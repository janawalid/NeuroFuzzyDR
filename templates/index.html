02

CLAHE + Fuzzy Orientation

BGR → LAB → CLAHE → RGB

CLAHE (Contrast-Limited Adaptive Histogram Equalization) is applied to the L channel in LAB colorspace, enhancing local contrast while suppressing noise amplification. Clip limit = 2.0, tile grid 8×8.

Fuzzy Orientation Correction — a Mamdani fuzzy system detects anatomically inverted fundus images using optic-disc x-position and corner notch ratio as inputs. If inversion confidence > 0.6, the image is flipped 180°.

03

EfficientNetB3 Backbone

300×300×3 → 1536-d feature vector

ImageNet-pretrained EfficientNetB3 (upgraded from B0) serves as the frozen feature extractor. Its compound-scaled architecture provides richer 300×300 feature maps versus B0's 224×224, yielding an expected +3–5% QWK improvement. 7 MBConv stages produce a 1280-d global average pooled representation.

04

Ordinal Regression Head

Cumulative-link · 4 boundary probabilities

Unlike softmax (which treats DR grades as independent), the cumulative-link ordinal head explicitly models the ordered structure (0<1<2<3<4). The Dense(4) → sigmoid layer outputs 4 cumulative probabilities P(Y≤k) for k=0..3, from which class probabilities are recovered as differences. Loss: binary_crossentropy on each boundary logit.

Ordinal regression4 boundary logitsBinary crossentropy
05

QWK Threshold Optimization

scipy.optimize · 4 free thresholds

Rather than using argmax over cumulative probabilities, 4 real-valued thresholds are optimized via scipy.optimize.minimize to directly maximize the Quadratic Weighted Kappa (QWK) on the validation set. This is the same metric used in the APTOS Kaggle competition. TTA (test-time augmentation with ±10°, ±20° rotations and centre zoom ×0.85) further boosts QWK.

Direct QWK maximization5-fold TTAscipy.optimize
06

Fuzzy Decision Adjustment

Confidence × Sharpness → Grade Δ

A second pure-NumPy Mamdani fuzzy system post-processes the raw grade using confidence (max class probability) and image sharpness (Laplacian variance) as inputs. Low confidence or blurry images may receive a conservative grade adjustment. Output is a crisp adjustment Δ ∈ [−1, +1], added to the raw grade and clamped to [0,4].

Mamdani inferenceConfidence inputSharpness inputGrade adjustment
07

Final DR Grade

Grade 0–4 · Clinically calibrated

The final output is an integer DR grade (0–4) with associated class probabilities and a fuzzy adjustment log. The pipeline respects the ordinal nature of DR severity throughout, from the cumulative-link head to the QWK-optimized thresholds to the fuzzy post-processor.

● Grade 0: No DR ● Grade 1: Mild ● Grade 2: Moderate ● Grade 3: Severe ● Grade 4: Proliferative

Methods

Technical Methodology

Two-Stage Training

Stage 1: frozen backbone, warm-up dense layers (LR=1e-3, 10 epochs). Stage 2: gradual backbone unfreezing with ReduceLROnPlateau and early stopping patience=5.

Class Weights

Inverse-frequency class weights computed via sklearn to counter the severe grade-0 dominance (≈49% of data). Ensures rare severe cases receive proportional gradient signal.

Anatomic Augmentation

Rotation ±20°, zoom ±10%, brightness [0.85–1.15], shift ±8%. Critically, NO horizontal/vertical flips or 90° rotations — anatomically invalid for fundus images.

Grad-CAM Visualization

Gradient-weighted Class Activation Mapping applied to the final convolutional block of EfficientNetB3 to produce pixel-level saliency maps highlighting DR lesion regions.

Pure-NumPy Fuzzy

Zero external fuzzy libraries. All membership functions (trimf, trapmf), rule evaluation (Mamdani), and defuzzification (centroid) implemented from first principles in NumPy.

Test-Time Augmentation

5-prediction TTA: original + ±10° rotation + ±20° rotation + centre zoom ×0.85. Mean pooling of cumulative probabilities before QWK-optimized threshold application.
